# Getting started

## Description

Nestjs Encoded API with swagger

## Installation

 Clone the repository
   git clone https://github.com/aglgit360/suzuki_cms_api_users.git

 Install dependencies
    npm install
 Copy env example file and set the configurations

## Running the app


# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
